# 3.3.repair.cicd.defects — inventory

## defect 1: keyrack extends absent role

### what
CI failed with:
```
BadRequestError: extended keyrack not found
{
  "path": ".agent/repo=bhuild/role=dispatcher/keyrack.yml",
  "from": "/home/runner/work/as-procedure/as-procedure/.agent/keyrack.yml"
}
```

### how (diff against origin/main)
```diff
# .agent/keyrack.yml was added with:
extends:
  - .agent/repo=bhrain/role=reviewer/keyrack.yml
  - .agent/repo=bhuild/role=dispatcher/keyrack.yml  # <-- not linked in CI
  - .agent/repo=ehmpathy/role=mechanic/keyrack.yml
```

### why (root cause)
the declapract upgrade added a keyrack.yml that extends bhuild/role=dispatcher, but:
- this repo doesn't use dispatcher features (task dispatch via radio channels)
- the role isn't linked in CI prepare step
- unnecessary extension for a simple library package

### fix
removed bhuild/role=dispatcher from extends list:
```yaml
extends:
  - .agent/repo=bhrain/role=reviewer/keyrack.yml
  - .agent/repo=ehmpathy/role=mechanic/keyrack.yml
```
